STLC:
 > Software Testing Life Cycle
 > STLC is the step-by-step process followed during software testing.
 > It defines the various activities and tasks to be performed at each stage of the testing process.
    - A sequence of specific activities conducted during the testing process to ensure software quality.
    - It provides a systematic approach to testing, ensuring that all aspects of the software are evaluated.
    - STLC consists of several phases, each with its own set of activities and deliverables.
    - The main phases of STLC include:
        1. Requirement Analysis
        2. Test Planning
        3. Test Case Development
        4. Environment Setup
        5. Test Execution
        6. Test Cycle Closure
