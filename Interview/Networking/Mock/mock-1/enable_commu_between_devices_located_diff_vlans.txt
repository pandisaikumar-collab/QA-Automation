q) how can we enable communication between devices located in different VLANs:

 - by default 2 VLANS can communicate within same switch if no routing is configured
 - to enable communication between different VLANS we need a Layer 3 device (router or layer 3 switch)
   this process is called inter-VLAN routing
 - inter-VLAN routing can be achieved using:
    1) Router-on-a-stick:
        - a single physical interface on the router is configured as a trunk port
        - sub-interfaces are created for each VLAN with their own IP address
        - switch port connected to the router is set as a trunk port
        - devices in different VLANs can communicate via the router

    2) Layer 3 Switch:
        - a layer 3 switch can perform routing functions
        - SVIs (Switch Virtual Interfaces) are created for each VLAN with their own IP address
        - devices in different VLANs can communicate directly through the layer 3 switch
    - routing protocols (like OSPF, EIGRP) can be used on layer 3 devices to manage routing between VLANs
