FTP: File transfer protocol:
 > used to transfer files between systems over a network
 > operates on the application layer (Layer 7) of the OSI model
 > uses separate control and data connections for communication
 > typically uses port 21 for control connection and port 20 for data connection
 > supports various commands for file operations like upload, download, delete and rename

Example:
 - A user wants to upload a file to a remote server using an FTP client.
 - The client establishes a control connection to the server on port 21.
 - The user issues an upload command, and the server responds with a request for data connection.
 - The client opens a data connection on port 20 and transfers the file to the server.
 - Once the transfer is complete, the data connection is closed, and the control connection remains open for further commands.
