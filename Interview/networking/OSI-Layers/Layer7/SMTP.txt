> SMTP stands for Simple Mail Transfer
> Protocol. It is a protocol used for sending and receiving email messages over the internet. SMTP operates at the Application Layer (Layer 7) of the OSI model. It is responsible for the transmission of email messages between mail servers and from email clients to mail servers. SMTP uses a set of commands and responses to facilitate the transfer of email messages

Example:
 Gmail sending an email to Yahoo Mail
  - When a user sends an email from their Gmail account to a Yahoo Mail account, the Gmail server uses SMTP to transmit the email message to the Yahoo Mail server.
  - The Gmail server establishes a connection with the Yahoo Mail server on port 25 (the default port for SMTP).
  - The Gmail server sends the email message along with the necessary headers (such as sender, recipient, subject, etc.) using SMTP commands.
  - The Yahoo Mail server receives the email message and stores it in the recipient's mailbox.
  - The recipient can then access their Yahoo Mail account to read the email sent from Gmail.
