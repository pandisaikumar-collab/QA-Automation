SNMP:
> Used to monitor and manage network devices,
> such as routers, switches, servers, and printers.
> Operates at the Application Layer (Layer 7) of the OSI model.
> Uses UDP as its transport protocol, typically on ports 161 (for general SNMP messages) and 162 (for SNMP trap messages).
> Allows network administrators to collect information about network performance, detect network faults, and configure network devices remotely.
> Utilizes a hierarchical structure of Management Information Bases (MIBs) to organize and define the data that can be accessed and managed.
> Common versions include SNMPv1, SNMPv2c, and SNMPv3, with SNMPv3 providing enhanced security features such as authentication and encryption.
> Widely used in network management systems (NMS) for monitoring network health and performance.
> Example use cases include monitoring bandwidth usage, tracking device uptime, and receiving alerts for network issues.
> SNMP agents are installed on network devices to collect and report data to SNMP managers.
> SNMP managers are software applications that communicate with SNMP agents to retrieve and analyze network data.

Keywords: Monitoring, Polling, Network Management, Traps
