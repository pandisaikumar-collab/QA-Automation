> Because one device can run many applications at the same time.
> Port numbers help to identify which application or service on a
  device should handle the incoming or outgoing data.
> They act like channels that direct the data to the correct application.
> Without port numbers, it would be difficult to manage multiple
  applications communicating over the network on the same device.

Example:
 - A web server typically uses port 80 for HTTP traffic and port 443 for HTTPS traffic.
 - An email server might use port 25 for sending emails (SMTP) and port 110 for receiving emails (POP3).
 - When a user accesses a website, the data is sent to port 80 or 443, ensuring it reaches the web server application.
 - Simultaneously, if the user is also checking their email, the email data is directed to ports 25 or 110, reaching the email server application.
