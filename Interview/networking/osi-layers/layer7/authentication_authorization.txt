> Authentication:
 > Authentication is the process of verifying the identity of a user,
 > Using credentials such as usernames and passwords, biometrics, or tokens.
 > It ensures that only authorized individuals can access certain resources or systems.

Example:
 - Login to a website using a username and password.
 - Using two-factor authentication (2FA) where a user must provide a password and a code

Keywords: Authentication and Authorization
    - Authentication: Verifying identity (e.g., login with username and password)
    - Authorization: Granting access to resources based on verified identity (e.g., user roles and permissions)
    - Roles example: Admin, User, Guest and RBAC (Role-Based Access Control)
